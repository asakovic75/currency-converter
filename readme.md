# Currency Converter API (Test Task)

## Стэк
- Node.js (v24.14.0)
- TypeScript, Express
- Supabase (DB)
- Swagger (Documentation)

## Особенности реализации
- **Двухуровневое кэширование**: 
  - Данные хранятся в БД в течение 24 часов.
  - Повторные запросы кэшируются в памяти на 5 минут.
- **Безопасность**: Использование httpOnly cookies для идентификации пользователей.
- **Интерфейс**: Реализован Dashboard для удобного тестирования функций.

## Демонстрация
Щтчет с описанием логики и скриншотами работы эндпоинтов находится в папке `/demo`.

## Запуск
1. `npm install`
2. Настройте `.env` файл
3. `npm run dev`

## Документация
Доступна по адресу: http://localhost:3000/api-docs
